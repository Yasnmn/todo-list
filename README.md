# To-Do List 📝

Uma aplicação simples de lista de tarefas feita com **Next.js** e **TypeScript**.

## Funcionalidades ✅

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Deletar tarefas da lista

## Tecnologias 🚀

- React
- Next.js
- TypeScript
- CSS Modules
